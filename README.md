Chat Application - React + TypeScript
A modern chat application built with React, TypeScript, and Tailwind CSS. Features user authentication, real-time messaging, and a responsive UI.

Features
-User Authentication - Login/Register system
-Real-time Chat - Send and receive messages instantly
-Modern UI - Tailwind CSS styling.
-Responsive Design - Works on desktop. Needs improvement so it can also be used on mobiles.

Composition Pattern for reusable and maintainable components

Stack
-Frontend: React 18 + TypeScript
-Styling: Tailwind CSS
-Validation: Zod
-Icons: Lucide React
-State Management: React Context + useReducer
-Forms: React Hook Form
-Date Handling: date-fns
